Simple Snake game. You can navigate using the arrow keys. Consume apples to score points, avoid hitting yourself. To quit, press ESC.

Try it out!

```
git clone https://github.com/thomas-pendragon/dablang
cd dablang
rake example[1]
```


Used libraries: SDL2.
